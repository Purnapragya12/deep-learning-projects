# Image Classification using Deep Learning (CNN)

## Project Overview
This project implements an image classification system using a Convolutional Neural Network (CNN) to classify images into different categories.

## Dataset
- CIFAR-10 dataset
- 60,000 color images in 10 classes
- Image size: 32x32x3

## Approach
- Data normalization
- CNN architecture with convolution, pooling, and dropout layers
- Model trained using Adam optimizer

## Results
- Achieved good classification accuracy within few epochs
- Visualized training and validation accuracy

## Tech Stack
- Python
- TensorFlow / Keras
- Matplotlib

## Use Case
Demonstrates real-world image classification using deep learning.
